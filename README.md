# Burger Store Management System
A desktop-based burger store management system built with Java and JavaFX, following MVC architecture principles.

## Overview
This project is a full-featured burger store management system developed using Java and JavaFX. 
The application supports customer ordering, custom burger creation, combo management, and staff administration with role-based access control.
The system follows the MVC (Model-View-Controller) architecture to ensure clean separation of concerns and maintainable code structure.

## Features
### Customer Features
- Custom burger creation with dynamic ingredient selection
- Combo meal creation (Burger + Side + Drink)
- Real-time price calculation
- Order confirmation and payment simulation

### Staff Features
- Staff login system
- Role-based access control (Manager / Staff)
- Ingredient price modification
- Recipe management (CRUD)
- Staff management (Add, Edit, Remove)

## Technical Highlights
- Implemented MVC architecture for clear separation of UI, business logic, and data layers
- Utilised JavaFX ObservableList and Property binding for reactive UI updates
- Applied object-oriented principles (abstraction, inheritance, polymorphism, composition)
- Designed enum-based ingredient system with dynamic pricing capability
- Implemented full CRUD functionality for staff and recipes
- Added input validation and exception handling for system robustness

## Project Structure
- Model: Business logic and data representation
- View: JavaFX UI components and scene management
- Controller: Handles user interactions and coordinates Model & View

## How to Run
1. Clone the repository
2. Open the project in an IDE
3. Ensure JavaFX is properly configured
4. Run the main application file
